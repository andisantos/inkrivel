# Bibliotecas necessárias

- SDL2 (https://www.libsdl.org/)
- SDL2_mixer
- SDL2_image
- stb_truetype (https://github.com/nothings/stb/blob/master/stb_truetype.h

# Para compilar
Entre no diretório `src`:
```
make
./build/bin/runner
```

# Para rodar os testes
Entre no diretório `src`:
```
make
make test
./build/test
```
